# 🎓 AI Learning Buddy

An AI-powered learning assistant built using **Streamlit** and **Google Gemini AI**. The application helps users understand any topic through simple explanations, real-life examples, quizzes, summaries, interview questions, and interactive AI responses.

---

## 🚀 Features

- Explain concepts in simple language
- Generate real-life examples
- Create quizzes with answers
- Advantages & Disadvantages
- Interview Questions
- Topic Summaries
- Ask Anything about the selected topic
- Interactive and beginner-friendly interface

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- Generative AI

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/shaina24-3/AI-Learning-Buddy.git
```

Move into the project directory

```bash
cd AI-Learning-Buddy
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 API Configuration

Create an environment variable named:

```
GEMINI_API_KEY
```

Assign your Google Gemini API key to it before running the application.

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

---

## 🎯 How to Use

1. Enter a topic.
2. Select an activity.
3. Click **Generate AI Response**.
4. Explore explanations, quizzes, summaries, interview questions, or ask any question related to the topic.
