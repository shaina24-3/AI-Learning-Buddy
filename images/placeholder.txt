# 🎓 Shaina's AI Learning Buddy

An interactive AI-powered learning assistant built with **Streamlit** and **Google Gemini AI** that helps users learn any topic through simple explanations, real-life examples, quizzes, summaries, interview questions, and more.

---

## 🚀 Live Demo

🔗

## 📌 Project Overview

Learning new concepts can often be overwhelming, especially for beginners. **Shaina's AI Learning Buddy** is designed to make learning interactive, personalized, and engaging using the power of Generative AI.

Users simply enter any topic, select the type of learning activity they want, and receive AI-generated responses tailored to their needs.

---

## ✨ Features

- 📚 Explain concepts in simple language
- 💡 Generate real-life examples
- 📝 Create quizzes with answers
- ⚖️ Advantages & Disadvantages
- 🎯 Interview preparation questions
- 📄 Topic summaries
- 💬 Ask anything related to the topic
- ⚡ Instant AI-generated responses
- 🌙 Modern dark-themed UI

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/shaina24-3/AI-Learning-Buddy.git
```

Move into the project directory

```bash
cd AI-Learning-Buddy
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure API Key

Create an environment variable named:

```
GEMINI_API_KEY
```

Assign your Google Gemini API key to it.

Example (Linux/macOS):

```bash
export GEMINI_API_KEY="YOUR_API_KEY"
```

Example (Windows PowerShell):

```powershell
$env:GEMINI_API_KEY="YOUR_API_KEY"
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will open in your browser.

## 📖 How to Use

1. Enter any learning topic.
2. Select an activity from the dropdown.
3. Click **Generate AI Response**.
4. Read the explanation, summary, quiz, or interview questions.
5. Continue exploring different topics.
